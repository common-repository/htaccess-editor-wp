== Description ==

A simple wordpress plugin to edit htaccess file safely.

.htaccess editor WP is very simple and lightweight plugin to edit you site's htaccess editor safely. You don't need your server's account credentials to edit your server's configuration. Now you can simply install this plugin and can edit your file safely. Remember, any false code in htaccess file can break your site. Don't worry. This plugin won't take any falsy code in your htaccess file. If you write any false code which could break your site down, .htaccess editor WP won't let you save that code in htaccess file. It will throw a error message.

This plugin is build with idea of letting non web developers edit their htaccess file safely. Need to do something with your server configurations? Google it, copy the code, install the plugin, put the code and save. If anything is wrong, plugin won't let you save the code.
